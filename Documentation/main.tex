\begin{abstract}
This report presents the complete design, engineering history, and implementation of a real-time spoken keyword detection system. The objective is to classify isolated 1-second audio recordings into one of five classes (\texttt{yes}, \texttt{no}, \texttt{up}, \texttt{down}, and \texttt{other}) with low latency and high accuracy to enable real-time game control via voice commands. We describe two distinct model implementations: a PyTorch-based model utilizing off-board Log-Mel Spectrogram features yielding a 5-Fold cross-validation accuracy of 98.37\%, and a TensorFlow/Keras-based model using a custom GPU-based onboard Mel-Spectrogram layer. Finally, we document the deployment of the selected model onto a desktop application utilizing an Electron-Vite frontend and a FastAPI backend running ONNX Runtime.
\end{abstract}
